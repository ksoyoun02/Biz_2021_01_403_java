내 프로젝트 github upload

1. github repository 생성
  http://github.com/ksoyoun02
  New repository

2. local 폴더 repository 선언
= local repository 만들기
    local 폴더에서 git bash here

    git init

3. config 설정
= local repository와 remote repository의 권한, 보안 설정
  
  git config --local user.name ksoyoun02
  git config --local user.email ksoyoun95@naver.com

4. remote 설정

   upload할 remote repository 주소 별명 설정

   git remote add origin https://github.com/ksoyoun02/Biz_2021_01_403_java.git
   
* github 업로드 금지 설정
로컬 폴더에 .gitignore 파일 생성
touch .gitignore
업로드 금지 목록 작성

5. 파일 add

  git add .

  = 현재 폴더의 모든 파일, 폴더를 upload 하기 위해서 압축하라

6. commit 설정
  현재 시점에 업로드한 정보의 comment를 설정하는 것

   git commit -m first

7. push

   git push -u origin master
